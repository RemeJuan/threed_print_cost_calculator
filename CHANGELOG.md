## [2.5.0] - 2026-03-03
### Added
- Added multi-material support for premium users, including material pickers and material list improvements.
- Added live material list updates while editing.
- Added app activity insights so we can better understand which features people use most and improve the app faster.
- Added tracking for key actions such as calculations and premium feature usage to help improve the experience.

### Changed
- Improved multi-material calculations and input handling.
- Improved material weight input behavior for smoother editing.
- Improved material usage handling so blank entries are accepted when appropriate.
- Updated material-related text across multiple languages.
- Updated supporting libraries to newer versions for better compatibility and stability.
- Improved app header styling for a more polished and consistent look.
- Improved how app data is structured internally for better long-term stability.
- Cleaned up version display formatting in Support to make app version details easier to read.

### Fixed
- Made unassigned material naming and behavior more consistent.
