description: Suggest improvements to AGENTS.md based on the work just completed
---

Review the work that was just completed, including:
- tasks performed
- mistakes made
- corrections applied
- repeated instructions or clarifications needed
- any friction or inefficiencies observed

Goal:
Identify updates that would improve future agent performance via AGENTS.md.

Instructions:
1. Identify patterns:
   - Where did the agent need repeated prompting?
   - Where were instructions unclear or missing?
   - What caused incorrect assumptions or wasted effort?

2. Extract improvements:
   - Add rules to prevent repeated mistakes.
   - Add constraints to avoid known failure modes.
   - Add preferences that were implicitly required but not documented.
   - Add process steps for recurring workflows.

3. Be strict:
   - Only suggest changes that provide clear, repeatable value.
   - Do not add noise or overfit to one-off scenarios.
   - Prefer generalizable rules over specific fixes.

4. Output format:
   - Proposed additions or updates to AGENTS.md
   - Each as a clean, copy-ready markdown block
   - Grouped by section (e.g. "Execution Rules", "Tooling", "Localisation", etc.)

5. Do not modify AGENTS.md directly.
   - Only propose changes.

6. Keep suggestions concise and actionable.
