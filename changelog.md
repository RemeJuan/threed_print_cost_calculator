# Changelog

All notable changes to **3D Print Cost Calculator** are documented in this file.

## 2026-04-02

### Fixed
- Resolved a calculation refresh bug where switching from a material with a positive cost to a zero-cost material did not recompute totals.

### Improved
- Added and refined the public changelog page rendering, including safer remote markdown loading and updated repository links.
- Disabled outbound links in the rendered changelog view to keep in-app navigation predictable.
